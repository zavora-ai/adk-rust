# Intra-Compaction Example

Demonstrates intra-invocation context compaction from ADK-Rust v0.7.0 — automatically summarizing older conversation events when the estimated token count exceeds a configurable threshold.

## What This Shows

- Configuring `IntraCompactionConfig` with token threshold, overlap count, and chars-per-token ratio
- Using `estimate_tokens()` to monitor estimated token usage across conversation events
- Wiring `LlmEventSummarizer` and compaction config into the `Runner`
- Overlap preservation: recent events are kept after compaction for continuity
- Coherence after compaction: the agent still references earlier context through the summary

## How Intra-Compaction Works

During a long conversation, the context window can grow beyond what the LLM can handle efficiently. Intra-compaction addresses this by:

1. **Estimating tokens** before each LLM call using a character-based heuristic (`total_chars / chars_per_token`)
2. **Triggering summarization** when the estimate exceeds `token_threshold`
3. **Preserving recent events** (controlled by `overlap_event_count`) so the agent retains immediate context
4. **Replacing older events** with a single summary event generated by the `LlmEventSummarizer`

### Configuration Settings

| Setting | Default | This Example | Description |
|---------|---------|-------------|-------------|
| `token_threshold` | 100,000 | 2,000 | Token count that triggers compaction |
| `overlap_event_count` | 10 | 3 | Recent events preserved after compaction |
| `chars_per_token` | 4 | 4 | Character-to-token ratio for estimation |

The low threshold in this example ensures compaction triggers quickly for demonstration purposes. In production, use higher values (50,000–100,000) based on your model's context window.

## Key APIs Demonstrated

```rust
use adk_core::IntraCompactionConfig;
use adk_core::intra_compaction::estimate_tokens;
use adk_agent::LlmEventSummarizer;
use adk_runner::{Runner, RunnerConfig};

// Configure compaction
let config = IntraCompactionConfig {
    token_threshold: 2000,
    overlap_event_count: 3,
    chars_per_token: 4,
};

// Create summarizer (uses an LLM to generate summaries)
let summarizer = Arc::new(LlmEventSummarizer::new(model.clone()));

// Wire into the runner
let runner = Runner::new(RunnerConfig {
    intra_compaction_config: Some(config),
    intra_compaction_summarizer: Some(summarizer),
    // ... other fields ...
})?;

// Estimate tokens for monitoring
let tokens = estimate_tokens(&events, 4);
```
